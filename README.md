# 🎓 Educare Coaching Management System

A modern full-stack Coaching Management System built to simplify the day-to-day management of coaching institutes. The application provides an intuitive dashboard to manage students, teachers, fee collection, salary payments, and coaching records from a single platform.

🔗 **Live Demo:** [Educare Coaching Management](https://educare-coaching-management.vercel.app/)

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Ali-khan-786/Educare_Coaching_Management.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_PUBLISHABLE_KEY=YOUR_SUPABASE_PUBLISHABLE_KEY
```

Run locally

```bash
npm run dev
```

# 📈 Scalability

This application was primarily developed to digitize and simplify the management of my personal coaching institute. The current feature set is intentionally focused on the workflows I use daily, such as student management, teacher records, fee collection, and salary tracking.

However, the architecture is designed to be easily extendable. With additional development, the system can be scaled into a comprehensive coaching management platform by introducing features such as:

- Multi-branch coaching support
- Role-based access control (Admin, Teachers, Receptionists)
- Attendance management
- Online fee payment integration
- Parent and Student portals
- Exam & Result Management
- Timetable Scheduling
- Assignment Management
- SMS & Email Notifications
- Analytics and Reporting
- Mobile Application Support
- Cloud Storage for Documents
- Backup & Data Export

The current implementation serves as a strong foundation that can be expanded to support coaching institutes of varying sizes and operational requirements.
